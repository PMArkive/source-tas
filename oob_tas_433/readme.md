Use SPT version https://github.com/YaLTeR/SourcePauseTool/releases/tag/12-2025

## Watch the run
- All demos are in `run/demos`.
- Traces are in `run/traces.`

## Play the scripts
- Make sure `spt_disable_render_delay 0` for some scripts (mainly 18 & e02).
- The last 02 script might only play when the resolution is 1280x720.
- All saves are in `run/saves`. Can set them to read-only to force playback consistency.
